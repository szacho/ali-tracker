# AliTracker
An easy tool for parcels management. It does not require signing up nor signing in - just add [AliTracker](https://alitracker.herokuapp.com/) to your bookmarks or save generated token.

### Technologies

 - React + Redux
 - Express.js + MongoDB
 - Sass
